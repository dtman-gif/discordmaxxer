Built on the work of

Discordmaxxer is a fork of these excellent open-source projects (all GPL-3.0):

  • Vesktop by Vendicated — the Electron shell, screenshare-with-audio, tray, and Vencord bundling pipeline. Discordmaxxer is a fork.
  • Vencord by Vendicated — ~169 of the Discord client modifications you see in Discordmaxxer are Vencord plugins.

And these libraries:

  • Electron — cross-platform desktop apps with JavaScript, HTML, and CSS
  • Electron Builder — packaging and auto-update infrastructure
  • arrpc — an open implementation of Discord's Rich Presence server
  • rohrkabel — C++ RAII Pipewire-API Wrapper (used for Linux screenshare audio)
